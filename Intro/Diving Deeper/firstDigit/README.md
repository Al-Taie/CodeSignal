**Find the leftmost digit that occurs in a given string.**

### Example

For:

```
inputString = "var_1__Int"
inputString = "q2q-q"
inputString = "0ss"
```

the output should be:

```
firstDigit(inputString) = '1'
firstDigit(inputString) = '2'
firstDigit(inputString) = '0'
```
